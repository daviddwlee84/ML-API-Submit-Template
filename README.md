# ML API Submit Template

## Getting Started

```bash
pip install -r requirements.txt
```

## Usage

### CLI

> Better used with [`pueue`](https://github.com/Nukesor/pueue)

```bash
python cli.py --help
```

### API

```bash
python api.py
```

http://localhost:8000/docs

### WebUI

```bash
python api.py
# in another terminal
streamlit run ui.py
```

http://localhost:8501/docs
